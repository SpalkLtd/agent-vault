## Prerequisites

- A running Agent Vault instance ([see installation guide](/installation)).
- [NanoClaw](https://nanoclaw.dev/) installed.

## 1. Create the agent

If you haven't logged in yet, you'll be guided through setup automatically.

```bash
agent-vault agent create nanoclaw
```

This prints an agent token and copies it to your clipboard.

## 2. Supply the token to NanoClaw

Set `AGENT_VAULT_TOKEN` and `AGENT_VAULT_ADDR` in NanoClaw's environment (or run NanoClaw under `agent-vault run -- ...`) so it can authenticate with Agent Vault.
